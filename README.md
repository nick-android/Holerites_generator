# Streamlit Payroll App

This is a Streamlit payroll application that reads employee data and biometric attendance PDFs, calculates payroll for a selected date range, and generates PDF payslips.

## Run locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run esesmio.py
   ```

## Deploy to Streamlit Cloud

1. Create a GitHub repository and push this project.
2. Go to https://streamlit.io/cloud and connect your GitHub account.
3. Create a new app using this repository.
4. Set the main file path to `esesmio.py`.
5. Deploy and share the public link.
